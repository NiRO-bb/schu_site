# schu_site

"ШУ-сайт" (ШУ - вымышленное имя)
Сайт, предоставляющий возможность получить случайный референс из базы данных для художника.
Хотелось сделать подобие сайта https://quickposes.com/ru

СТЕК ТЕХНОЛОГИЙ:
Языки: HTML, CSS, PHP
База данных: MySQL
